# Marketing Layout Skeletons

## skeleton: hero-cta
```html
<section class="hero">
  <div class="hero__eyebrow"><!-- Short hook or badge --></div>
  <h1 class="hero__title"><!-- Value prop headline --></h1>
  <p class="hero__sub"><!-- 1-sentence reinforcement --></p>
  <div class="hero__actions">
    <a class="btn btn--primary" href="#"><!-- CTA --></a>
    <a class="btn btn--ghost" href="#"><!-- Secondary CTA --></a>
  </div>
</section>
```

## skeleton: feature-grid
```html
<section class="features">

  <div class="feature-grid">
    <div class="feature-card">
      <div class="feature-icon"><!-- icon --></div>
      <h3><!-- Feature name --></h3>
      <p><!-- 1-2 sentence description --></p>
    </div>
    <!-- repeat 3–6 cards -->
  </div>
</section>
```
